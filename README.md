Version Control system is a system that records file changes over time to avoid saving multiple copies. It records all made changes with a full history track, like who made the change, when this was done, the reason etc. 
The main benefits of Version Control for developers include:
-	Easy tracking of all changes and edits that have a time stamp and record of who perform the change.
-	Easy changes back in case of mistakes by going back to any previous version.  
-	Work in Branches by developing new features separately and not affecting the main code.
-	Allows for collaborative work by working separately on a same project and not overwriting each other.  
-	Remote backup for keeping copies of the project online to prevent data loss even if computer fails.
